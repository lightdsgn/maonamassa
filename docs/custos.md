# 9. Custos e Orçamento

A estimativa de custos do projeto **Mão na Massa** considera os investimentos necessários para o desenvolvimento inicial da plataforma, infraestrutura de operação e manutenção contínua. Os valores abaixo representam uma projeção realista baseada nos requisitos do sistema e no mercado tecnológico atual.

---

## 9.1 Custos de Desenvolvimento

Incluem design, front-end, back-end, testes, integrações, criação de protótipos e documentação.

| Item                                                                          | Valor Estimado            |
| :---------------------------------------------------------------------------- | :------------------------ |
| Desenvolvimento do sistema (web + dashboards + módulos Cliente/Prestador/ADM) | R$ 25.000 a R$ 60.000     |
| Design de interface (UI/UX + identidade visual + protótipos no Figma)         | R$ 2.000 a R$ 5.000       |
| Testes, documentação e ajustes finais                                         | R$ 3.000 a R$ 6.000       |
| **Total de Desenvolvimento**                                                  | **R$ 30.000 a R$ 71.000** |

---

## 9.2 Custos de Infraestrutura

Representam os recursos necessários para manter a plataforma ativa e operante.

| Item                                                   | Mensal              | Anual                   |
| :----------------------------------------------------- | :------------------ | :---------------------- |
| Hospedagem em nuvem (Vercel/AWS/Firebase)              | R$ 80 a R$ 300      | R$ 960 a R$ 3.600       |
| Banco de dados + armazenamento (Firebase/MongoDB/AWS)  | R$ 50 a R$ 200      | R$ 600 a R$ 2.400       |
| Certificado SSL (quando não incluso)                   | ~R$ 12,50           | R$ 150                  |
| Serviços de e-mail e notificações (SendGrid/OneSignal) | R$ 30 a R$ 150      | R$ 360 a R$ 1.800       |
| Backup automatizado                                    | R$ 40 a R$ 100      | R$ 480 a R$ 1.200       |
| **Total de Infraestrutura**                            | **R$ 210 a R$ 750** | **R$ 2.550 a R$ 9.150** |

---

| Categoria                | Valor Estimado             |
| :----------------------- | :------------------------- |
| Desenvolvimento (único)  | R$ 30.000 a R$ 71.000      |

## 9.5 Observações Finais

- Os valores variam conforme equipe, complexidade e número de funcionalidades lançadas no início.
- A infraestrutura pode ser reduzida utilizando planos gratuitos de Vercel ou Firebase no começo.
- O custo operacional depende da necessidade de suporte e da escala de usuários.
- O primeiro ano tende a ter o maior investimento inicial (design + desenvolvimento + testes).
